# Como jogar!

    Tem que ter instalado o expo, melhor usando yarn

    * npm install --global expo-cli

    * yarn add expo


    Depois de instalar executar

    npm run start

    ou 

    yarn start

    Para jogar 🎮






# Tecnologias usadas 🐱‍💻
  
    * React Native
    * Expo
    * Styled Components
