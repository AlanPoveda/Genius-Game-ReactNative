# Como jogar!

    Tem que ter instalado o expo 

    * npm install --global expo-cli

    * npm install --save styled-components

    Depois de instalar executar

    npm run start

    Para jogar 🎮






# Tecnologias usadas 🐱‍💻
  
 * React Native
 * Expo
 * Styled Components
